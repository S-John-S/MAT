Test data-set
The structures for test data set were selected from the Protein DataBank keeping in mind to select all the possible variation of structures using the following critera:

1. All the structures having at least one ligand.
2. The Representative Structures at 90% Sequence Identity.
3. Stoichiometry in biological assembly: HETEROMER, HOMO-4-MER, HOMO-6-MER, HOMO-8-MER.
4. Structure type: Protein and Protein complexes with ligand, metal, DNA and RNA.
